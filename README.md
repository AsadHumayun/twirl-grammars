## Twirl Grammar 

> This is a project of mine to attempt writing a grammar for the Twirl templating language.

I do not expect it to be easy, but if it works and I
end up being able to come up with something effective,
it will hopefully be able to improve several people's
developer experience while working on the Play stack.

> See repository licence for licencing details.

Maintained by: [Asad Humayun](https://github.com/AsadHumayun)

Notes and Further Info about the project:

There are lots of various hard-coded regexes that need to be updated
in all places over this repository if any one of them is updated.

I will try to figure out a better way to handle this later, jsut trying
to get everything working at the moment since this is a lot of (new)
stuff for me!
